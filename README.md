# 📊 Sales Data Dashboard (Power BI)

## 📌 Project Overview

This project focuses on building an interactive Power BI dashboard to analyze and track online sales data. The dashboard provides meaningful insights into sales performance, profit trends, and customer behavior.

---

## 🚀 Key Features

* Interactive dashboard with filters and slicers (Quarter, State)
* KPI tracking for Sales Amount, Profit, and Quantity
* Category-wise and sub-category-wise analysis
* Monthly profit trend visualization
* Customer-level insights

---

## 🛠️ Tools & Technologies

* Power BI
* Excel (Data Source)
* DAX (Data Analysis Expressions)

---

## 📈 Project Learnings

* Built an interactive dashboard to generate actionable business insights from raw sales data
* Applied data cleaning and transformation techniques for better data modeling
* Created relationships between tables and performed data joins
* Used DAX to create calculated measures and KPIs
* Implemented slicers and filters for dynamic, user-driven analysis
* Designed multiple visualizations (bar chart, donut chart, line chart, area chart, map, etc.)
* Improved data storytelling through structured and visually appealing dashboard design
